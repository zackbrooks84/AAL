# Contributing

Thanks for your interest in contributing to AAL. You can help by opening issues, improving docs, adding new adversary or defender policies, or proposing new metrics.

1. Fork the repo and create your branch from `main`.
2. Create small focused PRs with clear descriptions.
3. Add tests for new functionality.
4. Use Python 3.10 or newer. Local run:
   - `python -m venv .venv && . .venv/bin/activate`
   - `pip install -U pip pytest`
   - `pytest -q`
